Detailed Description

Throws T:System.Text.EncoderFallbackException when an input character cannot be converted to an encoded output byte sequence. This class cannot be inherited.

Throws an exception because the input character cannot be encoded. Parameters specify the value and index position of the surrogate pair in the input, and the nominal return value is not used.

Parameters
charUnknownHighThe high surrogate of the input pair.
charUnknownLowThe low surrogate of the input pair.
indexThe index position of the surrogate pair in the input buffer.
Returns
None. No value is returned because the M:System.Text.EncoderExceptionFallbackBuffer.Fallback(System.Char,System.Char,System.Int32) method always throws an exception.
Exceptions
T:System.Text.EncoderFallbackExceptionThe character represented by charUnknownHigh and charUnknownLow cannot be encoded.
T:System.ArgumentOutOfRangeExceptionEither charUnknownHigh or charUnknownLow is invalid. charUnknownHigh is not between U+D800 and U+DBFF, inclusive, or charUnknownLow is not between U+DC00 and U+DFFF, inclusive.
