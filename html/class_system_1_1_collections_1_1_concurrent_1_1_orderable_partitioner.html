Parameters
keysOrderedInEachPartitionIndicates whether the elements in each partition are yielded in the order of increasing keys.
keysOrderedAcrossPartitionsIndicates whether elements in an earlier partition always come before elements in a later partition. If true, each element in partition 0 has a smaller order key than any element in partition 1, each element in partition 1 has a smaller order key than any element in partition 2, and so on.
keysNormalizedIndicates whether keys are normalized. If true, all order keys are distinct integers in the range [0 .. numberOfElements-1]. If false, order keys must still be distinct, but only their relative order is considered, not their absolute values.
