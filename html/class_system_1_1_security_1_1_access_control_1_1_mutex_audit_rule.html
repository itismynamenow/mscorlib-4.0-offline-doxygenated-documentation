Initializes a new instance of the T:System.Security.AccessControl.MutexAuditRule class, specifying the user or group to audit, the rights to audit, and whether to audit success, failure, or both.

Parameters
identityThe user or group the rule applies to. Must be of type T:System.Security.Principal.SecurityIdentifier or a type such as T:System.Security.Principal.NTAccount that can be converted to type T:System.Security.Principal.SecurityIdentifier.
eventRightsA bitwise combination of T:System.Security.AccessControl.MutexRights values specifying the kinds of access to audit.
flagsA bitwise combination of T:System.Security.AccessControl.AuditFlags values specifying whether to audit success, failure, or both.
Exceptions
T:System.ArgumentOutOfRangeExceptioneventRights specifies an invalid value.-or- flags specifies an invalid value.
T:System.ArgumentNullExceptionidentity is null. -or- eventRights is zero.
T:System.ArgumentExceptionidentity is neither of type T:System.Security.Principal.SecurityIdentifier nor of a type such as T:System.Security.Principal.NTAccount that can be translated to type T:System.Security.Principal.SecurityIdentifier.
