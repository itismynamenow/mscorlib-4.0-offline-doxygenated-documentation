Initializes a new instance of the T:System.Security.Cryptography.X509Certificates.X509BasicConstraintsExtension class. Parameters specify a value that indicates whether a certificate is a certificate authority (CA) certificate, a value that indicates whether the certificate has a restriction on the number of path levels it allows, the number of levels allowed in a certificate's path, and a value that indicates whether the extension is critical.

criticaltrue if the extension is critical; otherwise, false.
