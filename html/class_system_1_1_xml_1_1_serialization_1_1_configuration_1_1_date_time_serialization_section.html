Determines XML serialization format of T:System.DateTime objects.

Enumerator
Default 

Same as Roundtrip.

Roundtrip 

The serializer examines individual T:System.DateTime instances to determine the serialization format: UTC, local, or unspecified.

Local 

The serializer formats all T:System.DateTime objects as local time. This is for version 1.0 and 1.1 compatibility.
