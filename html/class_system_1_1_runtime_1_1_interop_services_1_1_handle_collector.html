Initializes a new instance of the T:System.Runtime.InteropServices.HandleCollector class using a name, a threshold at which to begin handle collection, and a threshold at which handle collection must occur.

Parameters
nameA name for the collector. This parameter allows you to name collectors that track handle types separately.
initialThresholdA value that specifies the point at which collections should begin.
maximumThresholdA value that specifies the point at which collections must occur. This should be set to the maximum number of available handles.
Exceptions
T:System.ArgumentOutOfRangeExceptionThe initialThreshold parameter is less than 0.-or-The maximumThreshold parameter is less than 0.
T:System.ArgumentExceptionThe maximumThreshold parameter is less than the initialThreshold parameter.
