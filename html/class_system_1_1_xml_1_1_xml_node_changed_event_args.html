XmlNodeChangedAction System.Xml.XmlNodeChangedEventArgs.Action => action

Gets a value indicating what type of node change event is occurring.

Returns
An XmlNodeChangedAction value describing the node change event.XmlNodeChangedAction Value Description Insert A node has been or will be inserted. Remove A node has been or will be removed. Change A node has been or will be changed. The Action value does not differentiate between when the event occurred (before or after). You can create separate event handlers to handle both instances.
