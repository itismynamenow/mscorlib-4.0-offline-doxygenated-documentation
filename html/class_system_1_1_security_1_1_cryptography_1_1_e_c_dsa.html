static

Creates a new instance of the specified implementation of the Elliptic Curve Digital Signature Algorithm (ECDSA).

Parameters
algorithmThe name of an ECDSA implementation. The following strings all refer to the same implementation, which is the only implementation currently supported in the .NET Framework:- "ECDsa"- "ECDsaCng"- "System.Security.Cryptography.ECDsaCng"You can also provide the name of a custom ECDSA implementation.
Returns
A new instance of the specified implementation of this class. If the specified algorithm name does not map to an ECDSA implementation, this method returns null.
Exceptions
T:System.ArgumentNullExceptionThe algorithm parameter is null.
