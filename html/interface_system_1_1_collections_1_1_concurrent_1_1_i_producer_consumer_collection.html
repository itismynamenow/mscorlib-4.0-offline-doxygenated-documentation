Detailed Description

Defines methods to manipulate thread-safe collections intended for producer/consumer usage. This interface provides a unified representation for producer/consumer collections so that higher level abstractions such as T:System.Collections.Concurrent.BlockingCollection`1 can use the collection as the underlying storage mechanism.
