System.Security.Cryptography.HMACRIPEMD160.HMACRIPEMD160 ( byte []  key)

Initializes a new instance of the T:System.Security.Cryptography.HMACRIPEMD160 class with the specified key data.

Parameters
keyThe secret key for T:System.Security.Cryptography.HMACRIPEMD160 encryption. The key can be any length, but if it is more than 64 bytes long it is hashed (using SHA-1) to derive a 64-byte key. Therefore, the recommended size of the secret key is 64 bytes.
Exceptions
T:System.ArgumentNullExceptionThe key parameter is null.
