Detailed Description

Represents a combination of a user’s identity and an access mask.

Represents a combination of a user's identity and an access mask. An T:System.Security.AccessControl.AuditRule object also contains information about how the rule is inherited by child objects, how that inheritance is propagated, and for what conditions it is audited.
