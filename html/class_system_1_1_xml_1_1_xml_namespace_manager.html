Adds the given namespace to the collection.

Parameters
prefixThe prefix to associate with the namespace being added. Use String.Empty to add a default namespace. NoteIf the T:System.Xml.XmlNamespaceManager will be used for resolving namespaces in an XML Path Language (XPath) expression, a prefix must be specified. If an XPath expression does not include a prefix, it is assumed that the namespace Uniform Resource Identifier (URI) is the empty namespace. For more information about XPath expressions and the T:System.Xml.XmlNamespaceManager, refer to the M:System.Xml.XmlNode.SelectNodes(System.String) and M:System.Xml.XPath.XPathExpression.SetContext(System.Xml.XmlNamespaceManager) methods.
uriThe namespace to add.
Exceptions
