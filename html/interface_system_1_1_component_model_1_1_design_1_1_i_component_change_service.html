Detailed Description

Provides an interface to add and remove the event handlers for events that add, change, remove or rename components, and provides methods to raise a E:System.ComponentModel.Design.IComponentChangeService.ComponentChanged or E:System.ComponentModel.Design.IComponentChangeService.ComponentChanging event.
