Detailed Description

Contains a cache of XML Schema definition language (XSD) and XML-Data Reduced (XDR) schemas. The T:System.Xml.Schema.XmlSchemaCollection class class is obsolete. Use T:System.Xml.Schema.XmlSchemaSet instead.
